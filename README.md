financial-ratio
==========================

A module library that provides a set of APIs to calculate financial ratios.

Some implemeted APIs: 

- Security ratios, 
- [x] Beta ratio
- [x] Correlation ratio
- [x] Q ratio 

- Accounting ratios,
- [x] Asset Management ratios
- [x] Debt Management ratios
- [x] Liquidity ratios
- [x] Market Value ratios
- [x] Profitability ratios
