financial-ratio
==========================

A module library that provides a set of APIs to calculate financial ratios.

Some implemeted APIs: 

- Security ratios, 
- [x] beta ratio
- [x] Correlation ratio

- Accounting ratios,
- [x] AssetManagement ratio
- [x] DebtManagement ratio
- [x] Liquidity ratio
- [x] MarketValue ratio
- [x] Profitability ratio
